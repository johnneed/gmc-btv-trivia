## Step 6 — Simulated 401 (manual / integration test)

Set up MSW or a local proxy that returns `HTTP 401` for `GET /games`. Start the dev server and open the app.

Expected: The player sees "Sign in to view this content" — not a blank page, not "Something went wrong."

## Step 7 — Simulated 500 (manual / integration test)

Configure the mock to return `HTTP 500` for `GET /games`. Reload the app.

Expected: The player sees "Something went wrong. Try again." and a "Try again" button. Clicking the button re-triggers the fetch.

---

## Step 8 — Simulated timeout (manual / integration test)

Configure the mock to delay response beyond 10 seconds (or use `setTimeout` in a service worker). Reload the app.

Expected: Within 10–11 seconds, the loading indicator disappears and an error message appears. The request does not hang indefinitely.
