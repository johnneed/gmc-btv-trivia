## PHP Class: `Trail_Trivia_Shortcode`

Location: `wp-plugin/trail-trivia/includes/class-shortcode.php`

Responsibility: Register the `[trail_trivia]` shortcode and enqueue player assets when the shortcode is rendered.

```
Methods:
  register(): void
    - Calls add_shortcode('trail_trivia', [$this, 'render'])
    - Called from the 'init' action in trail-trivia.php

  render(): string
    - Called by WordPress when [trail_trivia] is found in content
    - Calls enqueue_assets()
    - Returns '<div id="trail-trivia-root"></div>'
    - Must RETURN (not echo) — WordPress shortcode contract

  private enqueue_assets(): void
    - wp_enqueue_script('trail-trivia-player', player_js_url, [], TRAIL_TRIVIA_VERSION, true)
    - wp_add_inline_script('trail-trivia-player', config_js, 'before')
    - wp_enqueue_style('trail-trivia-player', player_css_url, [], TRAIL_TRIVIA_VERSION)
```

## `window.trailTriviaConfig` Shape

Injected by `wp_add_inline_script()` before the player JS runs:

```json
{
  "apiBase": "https://site.example/wp-json/trail-trivia/v1",
  "nonce":   "abc123..."
}
```

| Field | Source | Purpose |
|---|---|---|
| `apiBase` | `rest_url('trail-trivia/v1')` | Full URL of the WP REST API namespace |
| `nonce` | `wp_create_nonce('wp_rest')` | WordPress REST nonce; valid 12 hours |

The React app reads this at startup via `window.trailTriviaConfig?.apiBase` with fallback to `VITE_API_BASE_URL` (Phase 3 implementation, unchanged in Phase 6).
