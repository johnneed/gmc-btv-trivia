## Step 8 — ARIA automated audit

Requires Chrome for Testing + matching ChromeDriver. Install once:
```bash
npx browser-driver-manager install chrome
```

Then audit:
```bash
npm run build

CHROME_PATH="$HOME/.browser-driver-manager/chrome/mac_arm-150.0.7871.24/chrome-mac-arm64/Google Chrome for Testing.app/Contents/MacOS/Google Chrome for Testing"
CHROMEDRIVER_PATH="$HOME/.browser-driver-manager/chromedriver/mac_arm-150.0.7871.24/chromedriver-mac-arm64/chromedriver"

npx serve dist -p 4321 &
SERVE_PID=$!
sleep 3

npx axe http://localhost:4321 --exit \
  --chrome-path "$CHROME_PATH" \
  --chromedriver-path "$CHROMEDRIVER_PATH" \
  --load-delay 2000 --timeout 60
AXE_EXIT=$?

kill $SERVE_PID 2>/dev/null
echo "axe exit: $AXE_EXIT"
# Expected: 0 (zero violations found)
```
