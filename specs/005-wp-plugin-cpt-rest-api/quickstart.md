## Step 6 — Create a game as admin, verify round-trip (SC-002)

```bash
# Get a nonce for testing
NONCE=$(wp --allow-root eval 'echo wp_create_nonce("wp_rest");')

QUESTIONS_JSON='[
  {"questionText":"Q1","choices":[{"text":"A"},{"text":"B"},{"text":"C"},{"text":"D"}],"correctAnswerIndex":0,"answerText":"Because A."},
  {"questionText":"Q2","choices":[{"text":"A"},{"text":"B"},{"text":"C"},{"text":"D"}],"correctAnswerIndex":1,"answerText":"Because B."},
  {"questionText":"Q3","choices":[{"text":"A"},{"text":"B"},{"text":"C"},{"text":"D"}],"correctAnswerIndex":2,"answerText":"Because C."},
  {"questionText":"Q4","choices":[{"text":"A"},{"text":"B"},{"text":"C"},{"text":"D"}],"correctAnswerIndex":3,"answerText":"Because D."},
  {"questionText":"Q5","choices":[{"text":"A"},{"text":"B"},{"text":"C"},{"text":"D"}],"correctAnswerIndex":0,"answerText":"Because A again."}
]'

GAME=$(curl -s -X POST http://localhost/wp-json/trail-trivia/v1/games \
  -H "Content-Type: application/json" \
  -H "X-WP-Nonce: $NONCE" \
  --cookie "$(wp --allow-root eval 'echo "wordpress_logged_in_" . COOKIEHASH;')=..." \
  -d "{\"title\":\"CDT Quiz\",\"status\":\"published\",\"publishDate\":$(date +%s000),\"questions\":$QUESTIONS_JSON,\"tags\":[\"hiking\"]}")

echo "$GAME" | jq '{id, title, status, publishDate, tagCount: (.tags | length), questionCount: (.questions | length)}'
# Expected: id is a UUID, title = "CDT Quiz", status = "published", tagCount = 1, questionCount = 5

GAME_ID=$(echo "$GAME" | jq -r '.id')

# Verify it appears in the public list
curl -s http://localhost/wp-json/trail-trivia/v1/games | jq 'length'
# Expected: 1

# Verify publishDate is an integer
echo "$GAME" | jq 'type_of(.publishDate) == "number"'
# Expected: true
```

## Step 10 — Settings endpoints

```bash
# Read settings
curl -s http://localhost/wp-json/trail-trivia/v1/settings \
  -H "X-WP-Nonce: $NONCE" | jq '{gamesPerPage, version}'
# Expected: gamesPerPage is an int, version is "1.0.0"

# Update gamesPerPage
curl -s -X PUT http://localhost/wp-json/trail-trivia/v1/settings \
  -H "Content-Type: application/json" \
  -H "X-WP-Nonce: $NONCE" \
  -d '{"gamesPerPage": 20}' | jq '.gamesPerPage'

# Reject version field (SC-007a)

```
